# [Terraform](https://www.terraform.io/) script launches ec2 instance on aws


## How to run?

Create a [terraform.tfvars] file in the project directory & provide values to variables from the variables.tf file


## Usage

```terraform
terraform init
terraform validate
terraform apply
```
